OpenAI 周一宣布与微软达成新协议，结束了双方围绕 OpenAI 在 AWS 上销售产品的潜在法律冲突。这份被业界估值约 500 亿美元的合作调整，将让 OpenAI 在亚马逊云上提供 GPT-5 及未来模型的企业级服务。

此前 OpenAI 与微软的独家合作条款让 Azure 长期作为 OpenAI 模型的唯一商业云分销渠道。随着 Anthropic 通过 AWS Bedrock 大举吃下企业市场，OpenAI 在云端市场的增长开始受到结构性限制。

作为交换，微软获得更高的 OpenAI 总收入分成（具体比例未披露），并保留对 OpenAI 模型的优先共享访问权。

Sam Altman 在内部备忘录中表示，这次调整解锁了 OpenAI 长期以来「第二大云市场」的增长瓶颈，将显著提升企业客户覆盖率。

## 对 AI 行业的影响

**OpenAI 与 Anthropic 的企业战进入正面对抗阶段。** Anthropic 通过 AWS Bedrock 在企业 AI 市场建立了优势，OpenAI 此前因独家与 Azure 绑定而落后。进入 AWS 后，双方将在同一渠道争夺同一批企业 IT 决策者。

**微软策略发生微妙转变。** 微软愿意「松绑」表面上是失去独占权，实际是用更高分成换取更广覆盖——微软自己的 Azure 增长很大程度依赖 OpenAI 模型，OpenAI 在 AWS 上的增长部分会通过分成回流。

**对国内 AI 公司是渠道警示。** 中国 AI 公司在国际市场分发依赖少数云厂商时，也会受类似限制。OpenAI 的处理方式提供了一个「对冲式」分发策略的案例。

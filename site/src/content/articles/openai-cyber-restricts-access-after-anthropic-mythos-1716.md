Stripe 宣布推出 Link，一款支持 AI 智能体自主完成支付的数字钱包，标志着支付基础设施正式向 AI 原生应用场景延伸。

Link 的核心创新在于其 API 设计允许 AI 代理在获得授权后自主发起支付、订阅和退款操作，无需人类干预。这将解锁 AI 智能体在电商、旅行、订阅服务等领域的自动交易场景。

## 对 AI 行业的影响

支付是 AI Agent 实现商业闭环的关键瓶颈。Stripe 的 Link 标志着金融基础设施正式承认 AI Agent 作为合法支付发起方。对于 AI Agent 的商业化路径，这是重要的基础建设突破——没有支付，AI Agent 的经济价值就无法兑现。
